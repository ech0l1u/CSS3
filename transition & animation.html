过渡效果

transition:要过渡效果的属性 过渡时间 速度效果 启动时间延迟
transition: all 1s
transition: width 2s
transition: background-color 3s
transition-property: width,transform;transition-duration: 3s

其实上面最后一个div里说明了一个问题，那就是如果要过渡效果的属性不止一个的话，就别用transition:这种缩写了，but，那个要过渡效果的属性的值可以用all啊。

animation

首先把animation定义好

@keyframes 动画名字
{	
    from {css属性名:值;}
    to {css属性名:值;}
}
	
@keyframes 动画名字
{	
    0% {css属性名:值;}
    N% {css属性名:值;}
    N% {css属性名:值;}
    N% {css属性名:值;}
    100% {css属性名:值;}
}

其实我不知道那个0%和100%是不是必要的，，，

animation: 动画名 动画持续时间 动画速度曲线 动画启动延迟 动画播放次数 是不是轮流反向播放
